# Cadastro em C

## PT-BR:
Simples código em C ao qual realiza buscas, listagem e adição de dados em um arquivo.

No código em questão, foi simulado um sistema de cadastro com menu, onde você possui opções de:
- Cadastro: Adiciona dados que o usuário insere, e é salvo no "cadastros.txt".
- Busca: Realiza busca por nome e imprime na tela, caso ache, os dados por Nome, Idade e Data de nascimento.
- Listar: Realiza uma listagem de todos os itens no "cadastros.txt", formatando em Nome, Idade e Data de nascimento.
  
## Instalacao:

Para compilar e executar o programa, você precisa primeiro instalar o arquivo cadastro.c.

Achar e Compilar no CMD:
```
cd C:\Users\... >> "pasta com cadastro.c"
```
```
gcc cadastro.c -o cadastro.exe
```
Executar:
```
./cadastro.exe
```

# EN:
A simple C code, which it's able to do searches, data entry and listing.

The code simulates a simple register system, where it features:
- Register: Adds user data to "cadastros.txt".
- Search: Search by name, if found, prints Name, Age and Birth.
- Listing: Prints all entries in "cadastros.txt", formatted by Name, Age and Birth.

## Installation

To be able to compile and run the program, you'll need to install "cadastro.c"

Find and Compile in CMD:
```
cd C:\Users\... >> "Folder with cadastro.c"
```
```
gcc cadastro.c -o cadastro.exe
```
Run:
```
./cadastro.exe
```
